Zsh w/ Prezto
=============

Files
-----

### General

- `zshrc`
    + Path: `$HOME/.zshrc`
    + `zsh` configuration file
    + Requires `$HOME/.aliases` to exist (see below)

- `zpreztorc`
    + Path: `$HOME/.zpreztorc`
    + `prezto` configuration file

### aliases

- `aliases`
    + Path: `$HOME/.aliases
    + Adds various useful Zsh (and Bash) aliases, including commands for
      schedule managing
